## English

A Claude Code skill that automates the entire App Store opportunity research pipeline — from discovering underserved niches to building working iOS prototypes.

Designed for **indie developers** and **startups** who want to find profitable app ideas and validate them quickly.

> Based on [froessell/app-store-opportunity-research](https://github.com/froessell/app-store-opportunity-research). Extended with Rork Max integration for native iOS prototype building.

### Installation

```bash
npx skills add https://github.com/froessell/app-store-opportunity-research --skill app-store-opportunity-research
```

### What It Does

```
App Store Charts → Competitor Deep-Dive → Gap Analysis → Top 3 Report → MVP PRD → Rork Prototype
```

1. **Discover niches** — Browse App Store charts to find underserved categories
2. **Analyze competitors** — Deep-dive into 5-8 competitor apps (ratings, revenue, complaints)
3. **Identify gaps** — Build feature comparison matrices to spot missing features
4. **Rank opportunities** — Produce a revenue-validated Top 3 opportunity report
5. **Write PRD** — Generate a 16-section MVP Product Requirements Document
6. **Build prototype** — Create a working native iOS prototype on [Rork](https://rork.com)

The entire pipeline runs **end-to-end in ~30-45 minutes**.

### Prerequisites

- **Chrome browser** with the [Claude in Chrome](https://chromewebstore.google.com/) extension (for App Store browsing)
- **Rork account** at [rork.com](https://rork.com) (optional, for prototype building)
- No API keys required — all research is done through live browser interaction

### Usage

Trigger the skill with phrases like:

- "find app opportunities"
- "app store research"
- "what app should I build"
- "research this app category"
- "find a gap in the app store"

A complete session produces:

| Output | Description |
|--------|-------------|

| Top 3 Opportunity Report | Ranked opportunities with revenue validation |

| Working prototype | Live on Rork with shareable URL |
| Go-to-market notes | Pricing, channels, launch plan |

## 日本語

App Storeの機会発掘パイプラインを自動化するClaude Codeスキルです。未開拓ニッチの発見からiOSプロトタイプの構築まで、すべてを一貫して実行します。

収益性の高いアプリアイデアを見つけて素早く検証したい**個人開発者**や**スタートアップ**向けに設計されています。

> [froessell/app-store-opportunity-research](https://github.com/froessell/app-store-opportunity-research) をベースに、Rork Maxによるネイティブ iOS プロトタイプ構築機能を追加しています。

### インストール

```bash
npx skills add https://github.com/froessell/app-store-opportunity-research --skill app-store-opportunity-research
```

### 機能概要

```
App Storeチャート → 競合分析 → ギャップ分析 → トップ3レポート → MVP PRD → Rorkプロトタイプ
```

1. **ニッチ発掘** — App Storeチャートを閲覧し、未開拓カテゴリを発見
2. **競合分析** — 5〜8個の競合アプリを詳細調査（評価数、収益、不満点）
3. **ギャップ特定** — 機能比較マトリクスを作成し、不足機能を発見
4. **機会ランキング** — 収益検証付きトップ3機会レポートを作成
5. **PRD作成** — 16セクションのMVP製品要件ドキュメントを生成
6. **プロトタイプ構築** — [Rork](https://rork.com)上でネイティブiOSプロトタイプを作成

パイプライン全体が**約30〜45分でエンドツーエンドに実行**されます。

### 前提条件

- [Claude in Chrome](https://chromewebstore.google.com/) 拡張機能がインストールされた **Chromeブラウザ**（App Store閲覧用）
- [rork.com](https://rork.com) の **Rorkアカウント**（プロトタイプ構築用、任意）
- APIキーは不要 — すべてのリサーチはライブブラウザ操作で行われます
